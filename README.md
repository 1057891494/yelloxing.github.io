### 快速开始
------
由于一些原因，项目没有明确的分成开发，预生产，生产等不同阶段的打包，因此你可以在项目根目录执行下面命令就可以直接在浏览器地址栏输入http://localhost:7070
```shell
npm run server
```
由于打包sass部分使用的是Ruby，其余部分使用的是node.js相关的工具，因此你首先要安装他们，具体安装不再说明，直接执行下面命令即可完成打包
```shell
npm install
```
```shell
npm start
```
