ECMAScript6 常用新特性
模板对象
等同于：var name = 'Your name is ' + first + ' ' + last + '.';
var name = `Your name is ${first} ${last}. `;
解构赋值
等同于：var data = $('body').data();house = data.house;mouse = data.mouse;
var { house, mouse} = $('body').data();
等同于把分割后的前三个数据方便给line1-3，把第五个数据给line5。
[line1, line2, line3, , line5] = '1n2n3n4n5n6n7'.split('n');
箭头函数
等同于：var _this = this; $('.btn').click(function(event){ _this.sendData(); });还有需要注意的是this问题。
$('.btn').click((event) =>{ this.sendData(); });

Symbols
首先是对Symbols的一些简单说明：
  • Symbol函数可以接受一个字符串作为参数，表示对Symbol实例的描述，主要是为了在控制台显示，或者转为字符串时，比较容易区分
  • 需要注意的是Symbol函数的参数只是表示对当前Symbol值的描述，因此相同参数的Symbol函数的返回值是不相等的
  • Symbol值不能与其他类型的值进行运算，会报错
  • 由于Symbol值都是不相等的，因此其可以作为对象属性名，此时，不能用点运算符
除了定义自己使用的Symbol值以外，ES6还提供了11个内置的Symbol值，指向语言内部使用的方法。
基本数据类型有6种：Undefined、Null、布尔值（Boolean）、字符串（String）、数值（Number）、对象（Object），这里新添加了一种：Symbol。
