理解函数
初识函数
每个函数都有一个length属性，和arguments的length不同的是，该属性不是实际传递的参数长度，而是函数声明时所需传递的形参个数。
函数的名称是定义时确定的（function name(){}），比如var getName=function name(){}，函数的名称是name而不是getName，别的也类似的不会改变函数名称。
没有块作用域，只有函数会创建新的作用域，而函数的上下文不是定义时确定的，是调用时确定的，不同的调用机制决定了不同的函数上下文：
  • 作为普通函数进行调用时，其上下文是全局对象
  • 作为方法调用时，其上下文是拥有改方法的对象
  • 作为构造器调用时，其上下文是新分配的对象
  • 通过函数的apply()或call()方法进行调用时，其上下文可以设置为任意值
闭包
作用域之外的所有变量，即便是函数声明之后的那些声明，也都包含在闭包中（见:闭包-example1）。 
/*闭包-example1*/
var outFun;
function outerFunction() {
    function innerFunction() {
        console.log(later);
    }
    outFun = innerFunction;
}
var later = '喉咙深处';
outerFunction();
outFun();
