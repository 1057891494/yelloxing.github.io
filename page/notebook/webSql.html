Web Sql Database
openDatabase：这个方法使用现有数据库或创建新数据库创建数据库对象。
transaction：这个方法允许我们根据情况控制事务提交或回滚。
executeSql：这个方法用于执行真实的SQL查询。
连接数据库
var dataBase = openDatabase("databaseName", "1.0", "测试数据库", 1024 * 1024, function () { });
  • 第一个参数代表数据库名称
  • 第二个参数代表版本号,目前为1.0
  • 第三个参数代表对数据库的描述
  • 第四个参数是用来设置数据的大小
  • 第五个参数是回调函数(可省略)
openDatabase方法打开一个已经存在的数据库，如果数据库不存在，它还可以创建数据库
执行sql语句
dataBase.transaction( function(tx) { tx.executeSql( "sql 语句", [], function(tx,result){//成功回调}, function(tx, error){//失败回调}); }); });
  • 第一个参数表示查询的字符串，使用的SQL语言是SQLite
  • 第二个参数代表插入到查询中问号所在处的字符串数据，是个数组
  • 第三个参数代表成功时执行的回调函数。返回两个参数：tx和执行的结果
  • 第四个参数代表一个失败时执行的回调函数。返回两个参数：tx和失败的错误信息
使用的SQL语言是SQLite，如果要创建表，执行对数据的增删改查，使用对应的sql遇见就可以了。
