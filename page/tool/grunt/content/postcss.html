使用grunt-postcss插件对CSS进行后处理

第一步：安装 

cnpm insall --save-dev grunt-postcss autoprefixer

第二步：Gruntfile.js插件配置部分 

postcss: {
        options: {
            processors: [
                require('autoprefixer')()
            ]
        },
        target: {
            src: 'style.css',
            dest: 'style.postcss.css'
        }
    }

不需要使用grunt.loadNpmTasks()方法加载autoprefixer
