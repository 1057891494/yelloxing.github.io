使用grunt-contrib-csslint插件对CSS进行语法检查

第一步：安装 

npm install --save-dev grunt-contrib-csslint

第二步：Gruntfile.js插件配置部分 

csslint: {
        options: {
            csslintrc: '.csslint'//使用.csslint里面的配置来确定如何检查
        },
        target: sourceStyleCsshint//检测的文件，是数组
    }

第三步：.csslint一般内容 具体可以查询手册，这里只是快速使用 

{
        "adjoining-classes":false,
        "box-sizing":false,
        "box-model":false,
        "compatible-vendor-prefixes": false,
        "floats":false,
        "font-sizes":false,
        "grandients":false,
        "important":false,
        "known-properties":false,
        "outline-none":false,
        "qualified-headings":false,
        "regex-selectors":false,
        "shorthand":false,
        "text-indent":false,
        "unique-headings":false,
        "universal-selector":false,
        "unqualified-attributes":false
    }
