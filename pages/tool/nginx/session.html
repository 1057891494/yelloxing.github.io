nginx 配置
  • session丢失问题
  • 配置文件 nginx.conf

第一步：发生情况

转发的时候，如果只是host、端口转换，则session不会丢失，如果路径也变化了，则需要设置cookie的路径转换，不然session会发生丢失。

第二步：解决方案

通过设置cookie的路径转换就可以解决了，具体例子如下： 

location /pre-url/ {
        proxy_pass http://serverProxy/new-url;
        proxy_cookie_path  /new-url /pre-url;
    }

由pre-url转发到new-url的时候这样设置就可以了。
